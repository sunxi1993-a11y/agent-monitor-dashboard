# 🚀 OpenClaw Agent监控面板

现代化的Agent监控仪表盘，支持多Agent状态显示、实时事件流和积分排名。

## 🌐 在线访问

部署到GitHub Pages后访问：
```
https://你的用户名.github.io/agent-monitor-dashboard/
```

## 📱 功能特性

### 🎨 三栏布局
- **左侧**：活动时间线
- **中间**：Agent状态卡片 + 积分排名
- **右侧**：实时事件流

### 🔧 技术特色
- 响应式设计（支持手机/平板/电脑）
- 实时数据更新
- 现代化卡片UI
- 流畅动画效果

## 🚀 快速部署

### 方法1：一键部署（推荐）
双击 `upload-to-github.bat`，按提示操作。

### 方法2：手动部署
1. 创建GitHub仓库：`agent-monitor-dashboard`
2. 上传本文件夹所有文件
3. Settings → Pages → 启用GitHub Pages
